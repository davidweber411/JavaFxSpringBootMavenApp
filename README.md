# Description

A simple running application based on JavaFX, Spring Boot and Maven.

### Used technologies

| Technology    | Version       |
|---------------|---------------|
| Spring Boot   | 3.2.4         |
| JavaFX        | 17.0.8        |
| Java          | 17            |
| Maven         | Maven Wrapper |
| Module system | Non modular   |

### Features

| Feature                   | Available | How to use                    |
|---------------------------|-----------|-------------------------------|
| Create an executable JAR. | Yes.      | Use the maven goal "package". |
